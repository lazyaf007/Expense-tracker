#Smart Expense Tracking & Budget Analysis

##  Project Overview

This project focuses on analyzing personal expense data to understand spending behavior and improve budgeting decisions. It helps identify high-spending areas and optimize financial planning.



## 🎯 Objectives

* Track daily expenses
* Analyze category-wise spending
* Monitor monthly spending trends
* Evaluate payment method usage



## 🛠️ Tools & Technologies

* Python (Pandas)
* Excel
* Tableau



## 📂 Dataset

* Type: Self-created dataset (simulating real-world expenses)
* Records: 100+ entries
* Features: Date, Category, Subcategory, Amount, Payment Mode



## ⚙️ Process

1. Data Creation & Cleaning (structured dataset in CSV format)
2. Data Transformation (converted dates, created month column)
3. Exploratory Data Analysis using Python
4. Dashboard Creation using Tableau



## 📊 Dashboard Features

* KPI: Total Expenses
* Monthly Expense Trend
* Category-wise Spending
* Payment Mode Analysis
* Expense Distribution (Pie Chart)



## 🔍 Key Insights

* Food and Bills are the highest expense categories
* UPI is the most frequently used payment method
* Spending trends vary across months
* Identified areas of unnecessary spending



## 📈 Outcome

The dashboard helps users understand their financial habits and make better budgeting decisions by reducing unnecessary expenses.
